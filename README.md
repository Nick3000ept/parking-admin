# СБ3 Паркинг — учёт выполненных работ

Веб-приложение для учёта работ по ~2000 помещениям паркинга. Подрядчики отмечают факт выполнения по своим назначениям, админ видит общую картину и при необходимости корректирует.

**Боевой URL**: https://nick3000ept.github.io/parking-admin/?t=<токен>

## Стек

- **Frontend**: один `index.html` (vanilla JS, без сборки) на GitHub Pages
- **Backend**: Google Apps Script Web App
- **БД**: Google Sheets (один файл, спецификация листов в [TZ.md](TZ.md))

## Документы

- **[TZ.md](TZ.md)** — техническое задание (структура листов, роли, UI, правила разработки)
- **[CLAUDE.md](CLAUDE.md)** — операционная шпаргалка (URL, ID, деплой, тестовые токены)
- **[DEPLOY_CHECKLIST.md](DEPLOY_CHECKLIST.md)** — что проверять перед каждым деплоем

## Роли

| Роль | ИД_подрядчик | Права |
|---|---|---|
| Подрядчик | свой латинский id (например `ovchi`, `spectr`) | Видит и отмечает только свои назначения |
| Админ | `admin` | Видит и отмечает любые работы за любого подрядчика |
| Наблюдатель | `viewer` | Видит всё, не может ничего менять |

Управление справочниками (работы / подрядчики / типы помещений) и назначениями (колонки `*СП` в листе `Главный`) — только через Google Sheets, в UI не выведено.

## Деплой

См. **[CLAUDE.md → Деплой](CLAUDE.md#деплой)**.

Кратко:
- **Фронт** → `git push` → GitHub Pages обновляется через ~1 мин
- **Бэк** → `clasp push --force && clasp update-deployment <DEPLOYMENT_ID>` (URL не меняется)
